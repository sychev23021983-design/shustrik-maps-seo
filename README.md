# shustrik-maps-seo

SEO-аудит, семантическое ядро и рабочая база для продвижения shustrik-maps.com в Google.

- **STATUS.md** — текущее состояние проекта: что сделано, что дальше (читать первым)
- **CLAUDE.md** — инструкции для Claude Code, если вы подключите его к этому репозиторию
- **audit-log.md** — история решений и изменений
- **semantic-core/** — CSV с категориями, товарными карточками, дублями и служебными страницами
- **lsi-keyword-bank.csv** — LSI-слова по типу карточки для написания контента

## Как обновлять
1. Изменили данные — обновите соответствующий CSV в `semantic-core/`
2. Обновите `STATUS.md` (дату и статус)
3. Добавьте запись в `audit-log.md`
4. `git add . && git commit -m "..." && git push`
